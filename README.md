# TP3 Flutter - Application de Quiz

Application mobile de quiz interactive développée avec Flutter et intégrée à Firebase.

## Fonctionnalités

- **Authentification** : Inscription et connexion avec Firebase Authentication
- **Quiz interactif** : Questions à choix multiples organisées par thèmes
- **Gestion des profils** : Photo de profil, préférences utilisateur et historique des scores
- **Système de scores** : Suivi des performances et classement
- **Panel d'administration** : Gestion des questions et thèmes (réservé aux admins)

## Technologies utilisées

- Flutter 3.5.0+
- Firebase (Auth, Firestore, Storage, Analytics)
- Provider (State management)

## Note sur la démo

Pour la démonstration, l'application a été testée principalement sur **l'émulateur Web**, notamment pour faciliter le **hosting** et obtenir des feedbacks permettant d'**enrichir Firebase** (Firestore, Storage) avec des données de test. Les tests initiaux de la fonctionnalité d'**importation des images** ont également été réalisés sur cette plateforme avant validation sur les autres supports.
